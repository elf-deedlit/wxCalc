# wxCalc
計算式の電卓
## 目的
久しぶりにwxPythonを触ったらバージョンが結構上がっていたのでVisualStudioCodeの勉強もかねて作成
## 使い方
* 対応はPython3のみ
* モジュールとしてwxPythonとsympyが必須
## 使い方
* 普通の(?)電卓と同じように計算式を入れてEnterキーを押して結果を表示するだけ
* クリアのショートカットはHomeキー
* 終了はESCキー
## 特徴
* 1+2*3=7になるよ
* 1/7+1/7+1/7+1/7+1/7+1/7+1/7=1になるよ
